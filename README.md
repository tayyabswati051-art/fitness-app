# 📱 Fitness Tracker App

A modern mobile fitness application built with **React Native (Expo)**.
The app allows users to explore exercises, view details, and track basic fitness information in a clean UI.

## 📲 Installation Guide

1. Open the APK download link
2. Download the file on your Android device
3. Open the APK file
4. Allow **“Install from Unknown Sources”** if prompted
5. Install and launch the app

---

## ✨ Features

* 🏋️ Exercise listing screen
* 📄 Detailed view for each exercise
* 🖼 Image-based UI
* ⚡ Smooth navigation using Expo Router
* 📱 Clean and responsive mobile design

---

## 🛠 Tech Stack

* **React Native (Expo)**
* **JavaScript**
* **Expo Router**
* **CSS / Styling (React Native Styles)**
